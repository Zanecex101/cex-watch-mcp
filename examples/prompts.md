# Example prompts

Once `cex-watch-mcp` is wired into your AI client, try these:

## Discovery

> "What exchanges does cex-watch monitor?"

> "Show me how this MCP server fetches data — is it really using public APIs only?"

## Listings

> "How many active spot pairs does Binance currently list?"

> "Give me 5 sample trading pairs from OKX."

> "Compare the spot pair counts across all 5 exchanges."

## Fees

> "Which CEX has the cheapest spot maker fee for new users right now?"

> "How does Coinbase's default fee compare to Binance's?"

## Status

> "Is the Kraken API healthy? What's the latency?"

> "Run a health check on all 5 exchanges and tell me if anything's down."

## Analysis-style (combining tools)

> "Show me the cheapest 3 exchanges by taker fee, and confirm their APIs are healthy."

> "Has Bybit's pair count gone up or down since yesterday's snapshot in this repo?"
